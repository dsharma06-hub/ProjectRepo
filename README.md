Overview:
The University Navigator Project is a web-based campus navigation system designed to help students and visitors easily explore university locations. It provides a simple interface along with voice-assisted navigation for better accessibility and user experience.

Features:
Campus information and navigation
Voice-based interaction
Multiple information pages
Simple and user-friendly interface

Technologies Used:
HTML5
CSS3
JavaScript

How to Run:
Download or clone the repository.
Open index.html in any web browser.
